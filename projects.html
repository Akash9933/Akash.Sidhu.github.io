Arena Style Battle Game

Using Java I created an Arena Style Battle Game for a school project. Using Object-Oriented Programming I tracked the players information, such as health, attack strength, experience, money and defense. Alongside the player information, I used Object-Oriented Programming to track information about the enemies and bosses in a simillar fashion. The game was designed for a Object-Oriented Modeling and Design course. The project was given full creativity to create any program we wished to creat while following guidelines the course required, such as the steps to creating an application. Alongside the project I created an Oral presentation showcasing the programs functionality alongside the guidelines the course required.
